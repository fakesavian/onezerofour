# OneZeroFour

A retro-terminal style interactive interface built with HTML, CSS, and JavaScript. The project features a minimalist, text-based design inspired by classic computer terminals.

## Features

- Retro terminal aesthetic with ASCII art elements
- Boot sequence and loading animations
- Interactive navigation system
- Multiple sections including:
  - The World (Districts & Architects)
  - The Game (Start & Load)
  - System navigation

## Structure

```
onezerofour/
├── index.html          # Main entry point
├── loading.html        # Loading screen
├── world.html         # The World section
├── game.html         # The Game section
├── districts.html    # Districts information
├── architects.html   # Architects information
├── exit.html        # Exit confirmation
├── styles/          # CSS files
│   └── main.css     # Main styles
└── scripts/         # JavaScript files
    └── main.js      # Main functionality
```

## Setup

1. Clone the repository:
```bash
git clone [repository-url]
```

2. Open `loading.html` in a web browser to start the experience.

## Design

The interface follows a minimalist, retro-terminal aesthetic featuring:
- Black background with white text
- ASCII art borders and decorations
- Monospace typography
- Numbered navigation options
- Boot sequence animations

## Navigation

- Main Screen → The World → Districts/Architects
- Main Screen → The Game → Start/Load
- Main Screen → Exit → Confirm/Return
